# Custom Maker 페이지 개발 완료 보고서

**작성일**: 2026년 5월 15일  
**개발자**: nomura (프론트·백엔드 개발자 역할)  
**프로젝트**: [human-bug-tier](https://github.com/nomura-kanziro/human-bug-tier.git)  
**목적**: tier-class를 기반으로 한 **완전한 커스텀 티어 메이커** 페이지 구현

---

## 1. 개발 개요

기존 `tier-class` 페이지의 구조와 스타일을 유지하면서, 사용자가 자유롭게 캐릭터를 배치하고 저장할 수 있는 **커스텀 티어 메이커**를 새로 개발하였습니다.

- **독립된 페이지**: `/custom-maker/index.html`
- **핵심 요구사항**: 드래그 앤 드롭, tier 간 상태 유지, 다운로드 기능, 초기화 안전성
- **상태 관리**: localStorage 기반 (새로고침·티어 전환 시에도 유지)

---

## 2. 구현된 주요 기능

### ✅ 핵심 기능
- tier 1~9 전체 자동 로드 및 화살표 전환 (`← 이전 티어 / 다음 티어 →`)
- tier-class의 모든 캐릭터를 풀에 자동 로드
- **드래그 앤 드롭 완벽 지원**
  - 풀 → 티어 이동
  - 티어 내부 자유 재정렬 (상하좌우, 두 캐릭터 사이 삽입)
  - 티어 → 풀 되돌리기
- **상태 영구 보존** (localStorage)
  - 티어 전환, 새로고침, 브라우저 종료 후에도 유지
- **완전 초기화** 버튼 (localStorage까지 초기화)

### ✅ 다운로드 기능
- **PNG 다운로드**: 1티어 → 9티어까지 **순차 자동 저장** (`tier-1.png` ~ `tier-9.png`)
- **PDF 다운로드**: 1~9티어를 **9페이지 PDF**로 저장
- **JSON 다운로드**: 현재 배치된 전체 티어 데이터를 사용자가 원하는 형식으로 저장
  ```json
  {
    "티어표 명단 목록": {
      "1티어": {
        "갑급": ["우류 타츠오미", ...],
        "을급": ["츠루기 시노부", ...]
      },
      ...
    }
  }
  ```

### ✅ UI/UX 개선
- 액션 버튼을 **티어 테이블 바로 아래**로 배치
- 다운로드 버튼 클릭 시 가로 드롭다운 메뉴 (PNG / PDF / JSON)
- 초기화 버튼 애니메이션 준비 완료 (추가 가능)

---

## 3. 기술적 해결사항 (주요 버그 수정 내역)

| 버그 | 원인 | 해결 방법 |
|------|------|-----------|
| 초기화 후 드래그 불가 | 새로 생성된 `.char`에 이벤트 리스너 미등록 | `enableDragAndDrop()` 재호출 + 위임 방식 적용 |
| 테이블 내부 정렬 안 됨 | `dragover` 조건에서 같은 존 내부 이동 차단 | 조건 제거 + `getDragAfterElement` 개선 |
| 풀 → 테이블 이동 후 정렬 깨짐 | 이벤트 리스너 중복 등록 | `_dndBound` 플래그 + document/tierList/pool 위임 방식 |
| 새로고침 시 복제 | `loadTierStateToDOM()`에서 `innerHTML = ''` 미실행 | 항상 드롭존을 먼저 비우고 복원 |
| tier 전환 시 상태 유실 | `saveCurrentTierState()` 호출 타이밍 오류 | 화살표 버튼에서 **미리 저장** 후 전환 |

## 5. 사용 방법

1. `custom-maker/index.html` 열기
2. 아래 풀에서 캐릭터를 드래그해서 원하는 티어에 배치
3. `← →` 버튼으로 티어 이동 (배치 상태 유지)
4. **다운로드** 버튼 클릭 → PNG / PDF / JSON 선택
5. **초기화** 버튼으로 전체 초기화
