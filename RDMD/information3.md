## 1. 작업 개요

기존 관리자 페이지에 **파란색 네비게이션 바**를 추가하고,  
**필터링 · 검색 · 정렬 · 페이지네이션** 기능을 완전 구현했습니다.

- **feat**: 파란 네비게이션 + 고급 필터 + 검색 + 정렬 + 페이지네이션
- **fix**: 페이지네이션 초기 로드 문제 해결 (25개 미만일 때도 항상 표시)

---

## 2. 추가된 주요 기능

### 2.1 파란색 네비게이션 바
- **위치**: 제목 바로 아래
- **구성**:
  - 왼쪽: `댓글 탐색 기능` + 4개 필터 버튼
  - 오른쪽: 검색 입력창 + 검색 버튼 + 신고 사유 select

### 2.2 필터 기능
| 필터 종류       | 설명                          |
|----------------|-------------------------------|
| 전체 댓글      | 기본 (모든 댓글 표시)         |
| 일반유저 댓글  | 일반 사용자만 표시            |
| 관리자 댓글    | Admin User만 표시             |
| 신고한 댓글    | 신고된 댓글만 표시            |

### 2.3 정렬 기능
- **최신순** (기본)
- **오래된순**

### 2.4 검색 기능
- 작성자 ID 또는 댓글 내용(제목+내용) 검색
- Enter 키 + 🔍 검색 버튼 모두 지원

### 2.5 신고 사유 필터
- 없음 (전체)
- 도배 및 테러행위
- 비방 및 모욕행위
- 광고형 댓글
- 기타

### 2.6 페이지네이션
- **한 페이지당 25개** 표시
- 이전/다음 버튼 + 페이지 번호
- 댓글이 25개 미만이어도 항상 페이지네이션 표시
- 필터·검색·정렬과 완전 연동

## 4. 사용 방법

1. 관리자 페이지 접속
2. 파란 네비게이션 바에서 원하는 필터 선택
3. 필요시 검색어 입력 후 **검색** 버튼 또는 Enter
4. **최신순/오래된순** select로 정렬
5. 페이지네이션으로 페이지 이동

**모든 필터·검색·정렬·페이지네이션은 실시간으로 연동**됩니다.
