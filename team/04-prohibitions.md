# 04 — 금지사항 (Do Not)

깨면 **머지 보류** 수준입니다.  
AI가 제안해도 **사람이 거절**하세요.

---

## 보안 · 비밀

| 금지 | 이유 |
|------|------|
| `.env` 커밋 | 유출 |
| 비밀번호·URI·JWT를 README/이슈/채팅에 붙여넣기 | 유출 |
| 관리 API를 인증 없이 열기 | 사이트 장악 |
| 비번 재설정 토큰 평문 DB 저장 | 탈취 시 치명 |
| 일반 유저 JWT에 `isAdmin` 임의 부여 | 권한 상승 |

## 경로 · 배포

| 금지 | 이유 |
|------|------|
| 내부 링크 `/admin/...` 절대 경로 남발 | GH Pages·하위 폴더 404 |
| 프론트에 `http://localhost:5000` 프로덕션 고정 | 배포 후 API 실패 |
| GH Pages만으로 로그인·게시판 “완성” 공지 | 잘못된 기대 |
| `npx serve -p 5000` 을 풀스택 대체 | API·clean URL 상실 |

## 코드 위생

| 금지 | 이유 |
|------|------|
| 요청 없는 전 저장소 포맷/리네임 | 리뷰 불가·충돌 |
| `node_modules` 수동 수정 | 재현 불가 |
| 공용 `common.js` / auth 미들웨어 무테스트 대수술 | 전 사이트 장애 |
| 디버그용 관리 백도어 방치 | 보안 |

## 프로세스

| 금지 | 이유 |
|------|------|
| **`RDMD/commit_history/{본인}.md` 미작성 상태로 커밋·푸시** | 이력 공백 · 팀 규칙 위반 |
| 팀원에게 프로덕션 시크릿 메신저 평문 전송 | 유출 |
| AI 출력을 검증 없이 강제 푸시 | 회귀 |
| `.agents` / `team` 규칙 무시하고 “일단 동작만” 머지 | 기술 부채 |

## 짧은 기억용

```
시크릿 넣지 마
관리 API 잠가
경로는 getBasePath
API는 getApiBase
테스트는 :5000
범위 밖 건드리지 마
커밋 전 commit_history 본인 md 써라
```
